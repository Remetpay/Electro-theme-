# Gadget GRV Premium Storefront Prototype

A polished, responsive frontend ecommerce concept for **Gadget GRV**, a futuristic electronics and smart gadgets store.

This is intentionally a frontend prototype that can later be adapted into Shopify sections/templates. It does **not** process real payments.

## Included

- Luxury dark-tech visual system with deep black/navy gradients, electric blue accents, purple glow, glassmorphism cards, rounded borders, and cinematic lighting.
- Sticky glass header with logo, navigation, search/account/cart actions.
- Conversion-focused homepage: hero, trust microcopy, trust bar, category cards, best sellers, featured deal, benefits, discovery cards, reviews, newsletter, and footer.
- Product page template: gallery, rating, pricing, sale badge, benefit bullets, quantity selector, Add to Cart, Buy Now, trust badges, accordions, urgency, and recommendations.
- Collection page template: hero, SEO copy, filters, sorting, product grid, ratings, quick add, sale badges, and compare-at pricing.
- Cart drawer prototype: free shipping progress, recommended add-on, trust cues, payment marks, and clear summary.
- Mobile-first responsive layout with premium desktop presentation.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Shopify adaptation notes

Recommended Shopify mapping:

- Homepage sections: hero, trust bar, category grid, featured collection, deal banner, benefits, discovery tiles, testimonials, newsletter.
- Product template: media gallery, product form, dynamic checkout button, metafield-backed accordions, recommendations.
- Collection template: collection banner, faceted filters, sort picker, product card snippets.
- Cart: drawer or cart template with free shipping threshold, add-ons, payment badges, and policy reassurance.

Replace placeholder images with licensed product photography or Shopify-hosted media before production.
